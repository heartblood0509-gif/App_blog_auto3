# Blog Pick 서사구조 템플릿 핸드오프 문서

> **이 문서의 용도**: 다른 Python + Gemini 프로젝트에 블로그 글쓰기 서사구조를 이식하기 위한 자료.
> 기존 글쓰기 기능이 있는 프로젝트에, 아래 템플릿 데이터와 프롬프트 로직을 주입하면 됨.

---

## 통합 가이드 (클로드 코드용)

### 해야 할 일
1. 아래 `BLOG_TEMPLATES` 딕셔너리를 Python 파일로 만들기 (예: `blog_templates.py`)
2. 아래 `build_title_prompt()`, `build_generation_prompt()` 함수를 프롬프트 빌더로 만들기
3. 기존 글쓰기 파이프라인에서 Gemini API 호출 전에 이 프롬프트를 조합하여 전달

### 작동 흐름
```
사용자가 템플릿 선택 (3종 중 택1)
    ↓
선택된 템플릿의 analysis_result + 주제 + 키워드 →  build_title_prompt() → Gemini → 제목 5~7개 추천
    ↓
사용자가 제목 선택
    ↓
analysis_result + 주제 + 키워드 + 선택된 제목 + 옵션 → build_generation_prompt() → Gemini → 블로그 글 생성
```

import google.generativeai as genai
import json

# Gemini 설정
genai.configure(api_key="YOUR_API_KEY")
model = genai.GenerativeModel("gemini-2.0-flash")

# 1) 템플릿 선택
template = BLOG_TEMPLATES["review"]  # 또는 "brand_info", "brand_intro"

title_prompt = build_title_prompt(
    analysis_result=template["analysis_result"],

    keywords="두피 관리, 탈모 예방 샴푸"
)

titles = json.loads(title_response.text)

generation_prompt = build_generation_prompt(
    analysis_result=template["analysis_result"],
    topic="환절기 두피 관리",

    selected_title=titles[0]["title"],
    persona="32살 남성 직장인, 탈모 고민 2년차",
    product_name="OO샴푸",
    product_advantages="귀리단백질 함유, 약산성",
    char_count_range="2500-3500",
)
blog_response = model.generate_content(generation_prompt)
print(blog_response.text)
