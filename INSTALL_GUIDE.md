# BlogPublisher 설치 가이드

AI 블로그 자동 생성 + 네이버 자동 발행 데스크톱 앱입니다.
Mac과 Windows를 지원합니다.

---

## 📥 1단계: 설치 파일 다운로드

1. [최신 릴리스 페이지](https://github.com/heartblood0509-gif/App_blog_auto3/releases/latest) 접속
2. 본인 OS에 맞는 파일 다운로드:
   - **🍎 Mac (Apple Silicon, M1/M2/M3/M4)**: `BlogPublisher-1.0.0-arm64.dmg`
   - **🍎 Mac (Intel)**: 현재 미지원 (요청 시 추가 예정)
   - **🪟 Windows**: `BlogPublisher-Setup-1.0.0.exe`

> 💡 다운로드 크기는 약 **280~300MB** 입니다. Python 서버 + 브라우저 자동화 엔진이 포함되어 있습니다.

---

## 🍎 Mac 설치

### 설치 방법
1. 다운로드한 **`.dmg`** 파일 더블클릭
2. 창이 뜨면 **BlogPublisher.app**을 **Applications** 폴더로 드래그
3. **Applications** 폴더에서 **BlogPublisher** 더블클릭

### ⚠️ "확인되지 않은 개발자" 경고가 뜨면

Mac은 Apple 인증되지 않은 앱을 기본적으로 차단합니다. **안전한 앱이니 아래 순서로 허용**해주세요:

1. 경고창이 뜨면 **"취소"** 클릭
2. Apple 메뉴 🍎 → **시스템 설정**
3. 좌측에서 **"개인정보 보호 및 보안"** 선택
4. 스크롤 내려서 **"BlogPublisher은(는) 확인되지 않은 개발자의 앱이므로 차단되었습니다"** 문구 옆의 **"무시하고 열기"** 클릭
5. 비밀번호 입력하거나 Touch ID 인증
6. 경고창이 다시 뜨면 **"열기"** 클릭

> ✅ 한 번만 허용하면 이후로는 바로 열립니다.

## 🪟 Windows 설치

### 설치 방법
1. 다운로드한 **`.exe`** 파일 더블클릭
2. 설치 마법사 안내대로 진행
3. 설치 완료 후 시작메뉴에서 **BlogPublisher** 실행

### ⚠️ "Windows Defender SmartScreen" 경고가 뜨면

1. **"추가 정보"** 클릭
2. **"실행"** 버튼 클릭
3. 설치 계속 진행

> ✅ 한 번만 허용하면 됩니다.

## 🚀 2단계: 최초 실행 (약 15초 대기)

앱을 처음 열면 **내부 엔진을 준비하는 시간**이 필요합니다:
- Python 서버 시작 (약 10초)
- Chromium 브라우저 자동 감지

**검은 화면이 잠깐 보여도 정상**입니다. 기다려주세요.

> 최초 한 번만 오래 걸리고, 이후에는 빠르게 실행됩니다.

---

## 🔑 3단계: 최초 설정 (10~15분 소요, 1회만)

앱을 효과적으로 쓰려면 **3가지 설정**이 필요합니다.

### 설정 1: Gemini API 키 입력 (필수)

**AI 글 생성을 위한 Google의 무료 API 키**가 필요합니다.

#### API 키 발급 방법
1. [Google AI Studio](https://aistudio.google.com/apikey) 접속
2. Google 계정으로 로그인
3. **"API 키 만들기"** 클릭
4. 생성된 키를 **복사** (AIza로 시작하는 긴 문자열)

#### 앱에 입력
1. BlogPublisher 앱의 **우측 상단 🔑 열쇠 아이콘** 클릭
2. 복사한 API 키를 **붙여넣기**
3. **"저장"** 클릭

> 💰 **비용**: 블로그/쓰레드 텍스트 생성은 **무료 플랜**으로 충분합니다.
> 이미지 생성을 쓰려면 유료 플랜 전환 필요 (선택사항).

### 설정 2: 네이버 계정 등록 (필수)

네이버 블로그 자동 발행을 위해 네이버 계정을 등록해야 합니다.

1. 우측 상단 **"네이버 계정"** 버튼 클릭
2. **"계정 추가"** 클릭
3. 입력:
   - 네이버 **아이디**
   - 네이버 **비밀번호**
   - **블로그 ID** (본인 블로그 URL의 ID 부분, 예: `blog.naver.com/myid` 면 `myid`)
   - 별명(선택, 구분용)
4. **저장**

> 🔒 **보안**: 비밀번호는 본인 PC에만 저장되며, 외부로 전송되지 않습니다.

### 설정 3: 2단계 인증(2FA)이 켜져 있다면

네이버 2단계 인증을 사용 중이면 자동 로그인이 어려울 수 있습니다. **앱 실행 후 네이버 자동화 창이 뜨면 직접 인증 완료 후** 계속 진행하세요.

## ✍️ 기본 사용법

### 블로그 글 자동 생성 & 발행

1. 앱 메인에서 **"블로그"** 선택
2. **분석 방식** 선택:
   - **템플릿 선택** (추천 후기형 등 7개 제공)
   - **레퍼런스 입력** (참고할 다른 블로그 글 URL 또는 텍스트)
3. 주제, 핵심 키워드 등 **세부 설정**
4. AI가 추천하는 **제목 3개 중 하나 선택**
5. **"글 생성"** 버튼 클릭 → AI가 마크다운으로 글 작성
6. 미리보기에서 내용 확인 (필요시 섹션 편집)
7. **"발행"** 버튼 클릭
8. 네이버 계정 선택, 이미지 배치, 금칙어 검증 후 **발행 시작**
9. 자동으로 브라우저가 뜨고 네이버 블로그에 글 업로드됨
10. 완료 후 블로그 URL 확인

## ❓ 자주 묻는 질문 (FAQ)

### Q. 앱 업데이트는 어떻게 받나요?

**자동으로 알려줍니다.** 새 버전이 출시되면:
1. 앱 실행 시 "새 버전 있어요" 알림이 자동으로 뜸
2. **"다운로드"** 클릭
3. **"재시작"** 클릭 → 자동 업데이트 완료

직접 할 일은 **없습니다**.

### Q. 자동 발행 중 캡챠가 뜨면?

- 네이버 첫 로그인에서 간혹 캡챠가 뜹니다
- 자동화 브라우저 창에서 **직접 캡챠 풀기** (최대 2분 대기)
- 해결 후 자동으로 계속 진행됨

### Q. "서버 연결 중..." 에러가 뜨면?

- 앱을 **완전히 종료 후 재실행** (⌘Q 또는 작업관리자에서 강제 종료)
- 그래도 안 되면 PC 재부팅

### Q. Gemini API 키는 어디에 저장되나요?

- **본인 PC의 브라우저 저장소(localStorage)**에만 저장됩니다
- 외부 서버로 전송되지 않습니다
- 앱 재설치 시 다시 입력 필요

### Q. 여러 네이버 계정으로 발행 가능한가요?

- 네, **여러 계정 등록 가능**합니다
- 발행할 때마다 원하는 계정 선택

### Q. 네이버가 블로그를 막을 위험이 있나요?

- 자동화 도구 사용은 **본인 책임**입니다
- 단기간 대량 발행은 피하세요 (권장: 계정당 1일 2~3개 이내)
- 본인 계정, 본인 PC에서 동작하므로 일반적 위험은 낮음

### Q. 앱을 완전히 삭제하고 싶어요

**Mac**:
1. Applications 폴더에서 BlogPublisher 휴지통으로 드래그
2. 저장된 데이터: `~/Library/Application Support/BlogPublisher/` 폴더 삭제

**Windows**:
1. 설정 → 앱 → BlogPublisher → 제거
2. 저장된 데이터: `%APPDATA%\BlogPublisher\` 폴더 삭제
